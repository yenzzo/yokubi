# Antes de começar

## Do que você precisa?

Este guia pressupõe que você está aprendendo hiragana e katakana e estudando o vocabulário da língua japonesa por conta; a forma mais eficiente de aprender o kana (katakana + hiragana) é através de [exercícios de repetição](https://kuuuube.github.io/kana-quiz/kana/index.html).

Este guia pressupõe que você saiba **a)** quais sistemas de escrita o idioma japonês utiliza, **b)** como um texto japonêsa se parece e **c)** o que é kanji. 

Este não é um guia de vocabulário; portanto o leitor terá de suplementar essa lacuna por conta própria.

Se você precisa de ajuda com outros tópicos, dê uma olhada no [loop](https://morg.systems/58465ab9).

**Verifique:** 置 Como esse caractere é exibido no seu dispositivo? Se não há uma linha vertical no lado esquerdo, seu navegador está utilizando uma fonte chinesa para exibir caracteres japoneses. Nesse caso, dependendo do seu sistema operacional, são duas as soluções mais prováveis: instalar fontes japonesas e/ou modificar as configurações de idioma do seu dispositivo/browser.

## Como usar este guia para aprender japonês

Em poucas palavras, este guia não passa duma introdução. O nosso lema é o seguinte: consumir conteúdo nativo é a única forma de aprender um idioma e se tornar fluente nele. Este guia não incentiva o leitor a memorizar os conceitos aqui expostos; por isso, não tente fazê-lo.

Uma semana é demasiado tempo para dedicar a um capítulo somente; um dia, de igual maneira, ainda é muito tempo, mesmo que você esteja, juntamente ao estudo de novos conceitos, revisando os antigos.

Cada capítulo aqui exposto aborda conceitos gramaticais básicos; você deve os ler o mais rápido possível — é muito importante que, através da aquisição prematura, os conceitos tenham tempo suficiente para amadurecer no seu subconsciente. Dessa forma, a revisão dos conceitos — que, a princípio, não foram propriamente absorvidos — adquire um caráter mais produtivo.

Em algum momento, este guia lhe dirá quando você deve começar a 'consumir' conteúdo japonês; no entanto, você é livre para tentar a qualquer momento.

Este guia lhe explica os conceitos gramaticais da língua japonesa para que você, eventualmente, possa decompô-la — logo, é de suma importância que você tente, frequentemente, ler conteúdo em japonês, assim revelando os seus pontos fracos. Dessa forma, e somente dessa forma, você é capaz de transformar exposicão à língua em aprendizado.

A forma mais eficiente de aprender vocabulário é [selecionando palavras de mídias que você aprecia](https://lazyguidejp.github.io/jp-lazy-guide/) e, posteriormente, memorizá-las com flashcards; isto é o que chamamos de _mineração_ ('mining', no original). O programa mais recomendado para fazer isso é o Anki, que utiliza um sistema de repetição espaçada, isto é, modifica a frequência com que as palavras aparecem baseado em quanto tempo você demora para esquecê-las. No início, você pode utilizar listas de frequência e decks compartilhados, como o [kaishi 1.5k](https://ankiweb.net/shared/info/1196762551).

Se o seu objetivo é _falar_ japonês, você deve, absolutamente, assistir/ouvir a conteúdos orais no idioma japonês — animes, notícias, pessoas —; caso contrário, correrá graves riscos de soar estranho quando tentar se comunicar.  Isso pode muito bem ser feito depois que você aprender a ler, mas você tem de se expor a isto (anime, música, filmes, doramas etc.) o mais rápido possível, do contrário terá de corrigir alguns erros chatos no futuro: dificuldade em assimilar sons muito parecidos, mas levemente diferentes, acento tonal irregular e pronúncia incorreta.

## "Ajuda\!"

Algumas coisas naturalmente levam tempo, então tente não se estressar muito com o seu progresso.

Se você ainda está perdido, ou voltou aqui porque está confuso quanto ao aprendizado:

- Comece aprendendo o hiragana e o katakana _fora_ deste guia.
- Comece a estudar vocabulário _fora_ deste guia.
- Tenha em mente que nem todos os kanjis, quando tomados isoladamente, representam palavras japonesas completas.
- Lembre-se que a língua japonesa não se trata, de modo algum, duma conversão dos caracteres portugueses para os caracteres (ou ideogramas) japoneses [nenhuma língua é assim, sendo honesto].
- Inúmeros termos não farão sentido se interpretados sob uma ótica alheia à japonesa.
- Você não pode aprender um idioma sem consumi-lo.
- Muitas coisas não farão sentido — até que você consuma materiais japoneses o suficiente.
- Continue seguindo em frente. Isto é o que lhe permite aprender o idioma.

Não memorize esse guia. Isso não vai funcionar. Na verdade, tentá-lo pode até ser maléfico para você.

## Ainda com dificuldade?

Não há como negar que todos aprendem igualmente quando o assunto é maestria, fluência, exposição e experiência no mundo real. No entanto, o mesmo não é verdade quando se trata de _estudo deliberado_. Este guia, como já deve ter percebido, requer do leitor estudo deliberado, e isso pode não funcionar para todos. Se após duas semanas de estudo nada funcionar, tente outros materiais; ler explicações diferentes sobre a mesma coisa pode torná-la mais clara.

Apenas certifique-se de não esgotar as suas energias tentando aprendê-las. Maestria só é obtida mediante experiência concreta, no mundo real; no caso do aprendizado de idiomas, isso significa ler e ouvir.
